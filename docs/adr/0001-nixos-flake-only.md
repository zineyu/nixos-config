# Commit to NixOS flake only

This repository started as a standalone Home Manager flake (`lib/mkHome.nix`), but that helper has been removed and the active build path is now `nixos-rebuild build --flake .#<hostname>`. The host registry in `hosts/default.nix` and the per-host `modules` field are consumed as NixOS modules, not as standalone Home Manager modules. We decided to remove the standalone Home Manager path and make the host registry a first-class NixOS module registry. This lets host-level overrides (keyd, greetd, hardware settings) live in one place and avoids the confusion of a seam that supports two different build targets.
